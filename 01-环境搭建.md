# 01 - 环境搭建

## 系统要求

### macOS
- macOS 10.14 (Mojave) 或更高版本
- 磁盘空间：至少 2.8 GB（不包括 IDE/工具）
- 工具：bash, curl, file, git 2.x, mkdir, rm, unzip, which, xz-utils, zip

### Windows
- Windows 10 或更高版本（64位）
- 磁盘空间：至少 1.32 GB（不包括 IDE/工具）

### Linux
- 64位 Linux 系统
- 磁盘空间：至少 600 MB（不包括 IDE/工具）

## 安装步骤

### 1. 安装 Flutter SDK

#### macOS/Linux
```bash
# 下载 Flutter SDK
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable

# 添加到 PATH
export PATH="$PATH:`pwd`/flutter/bin"
```

#### Windows
1. 下载 Flutter SDK zip 文件
2. 解压到 `C:\src\flutter`
3. 将 `C:\src\flutter\bin` 添加到系统 PATH

### 2. 运行 Flutter Doctor

```bash
flutter doctor
```

这个命令会检查你的环境并显示需要安装的组件。

### 3. 安装 Android Studio（Android 开发）

1. 下载并安装 [Android Studio](https://developer.android.com/studio)
2. 安装 Android SDK
3. 配置 Android 模拟器或连接真机

### 4. 安装 Xcode（iOS 开发，仅 macOS）

1. 从 App Store 安装 Xcode
2. 运行 `sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer`
3. 运行 `sudo xcodebuild -runFirstLaunch`

### 5. 安装 VS Code（推荐编辑器）

1. 下载并安装 [VS Code](https://code.visualstudio.com/)
2. 安装 Flutter 和 Dart 扩展

## 验证安装

```bash
flutter doctor -v
```

确保所有必要的组件都显示为已安装。

## 创建第一个项目

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

## 常见问题

### Flutter 命令未找到
确保 Flutter bin 目录已添加到 PATH 环境变量。

### Android SDK 未找到
在 Android Studio 中安装 Android SDK，并设置 `ANDROID_HOME` 环境变量。

### Xcode 命令行工具问题
运行 `sudo xcode-select --reset` 重置 Xcode 路径。
