# 11 - 图片资源使用

## 1. 本地图片（推荐）
1) 准备资源  
   - 创建目录：`assets/images/`  
   - 放入文件：`assets/images/logo.png`

2) 配置 `pubspec.yaml`  
   ```yaml
   flutter:
     assets:
       - assets/images/
   ```
   > 修改 `pubspec.yaml` 后需要 **Hot Restart**（热重启）。

3) 代码使用  
   ```dart
   Image.asset(
     'assets/images/logo.png',
     width: 120,
     height: 120,
     fit: BoxFit.cover,
   );
   ```

4) 多分辨率适配（可选）  
   ```
   assets/images/logo.png        // 基准
   assets/images/2.0x/logo.png
   assets/images/3.0x/logo.png
   ```
   - 文件名相同，放在 `2.0x/`、`3.0x/` 子目录，Flutter 会按设备 DPR 自动选择。

## 2. 常用参数
- `width` / `height`：显示尺寸
- `fit: BoxFit.cover / contain / fill / fitWidth / fitHeight`：裁剪或缩放模式
- `alignment`：对齐方式，默认 `Alignment.center`
- `color` / `colorBlendMode`：为图片着色（做单色图标时有用）

## 3. 网络图片
```dart
Image.network(
  'https://example.com/img.png',
  width: 120,
  height: 120,
  fit: BoxFit.cover,
  loadingBuilder: (context, child, progress) {
    if (progress == null) return child;
    return const SizedBox(
      width: 120,
      height: 120,
      child: Center(child: CircularProgressIndicator()),
    );
  },
  errorBuilder: (context, error, stack) => const Icon(Icons.error),
);
```

## 4. SVG / 矢量图（补充）
- 需引入第三方库：`flutter_svg`
  ```yaml
  dependencies:
    flutter_svg: ^2.0.0
  ```
- 使用：
  ```dart
  import 'package:flutter_svg/flutter_svg.dart';

  SvgPicture.asset(
    'assets/icons/logo.svg',
    width: 120,
    height: 120,
  );
  ```

## 5. 实用提示
- 只要尺寸：用 `SizedBox` 包裹 `Image`，或直接设定 `width/height`
- 要背景 + 尺寸：直接用 `Container(width/height + decoration)` 包裹
- 频繁加载同一图片：可配合 `CachedNetworkImage`（需第三方库）
- 调试路径问题：确认 `pubspec.yaml` 缩进与路径拼写；必要时 `flutter clean && flutter pub get`
